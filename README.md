# World Clock (nevermore time atlas)

A simple and elegant world clock application that displays the current time across multiple cities around the globe in real-time.

## Features

- Real-time clock updates every second
- Multiple time zones supported
- Popular cities included by default
- Select current cities and time zones
- Responsive design
- Mobile-friendly interface
- Clean UI

## Technologies Used
- HTML5: Structure of the web application.
- CSS3: Styling of the web application.
- JavaScript: Dynamic behavior and time updates.

## Open the Project

Simply open index.html in your browser.

No installation or dependencies required.
